# Public images

Drop marketing assets here. Files in this directory are served at `/images/<name>`.

## Required files

- `dashboard-preview.png` — Statistics dashboard screenshot shown inside the desktop screen mockup on the landing page (hero section of `resources/views/welcome.blade.php` and `welcome-production.blade.php`). Recommended ~1600×1000 (16:10), PNG or JPG.

If the file is missing the hero falls back to a placeholder icon automatically.
